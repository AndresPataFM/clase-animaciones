Propiedades del contenedor de Grid

Info 
                    
    display: grid | inline-grid;
    grid-template-rows:
    grid-template-columns:
    grid-template-areas: 
        "nombreGridArea | . | none | ..." /* un . es un area vacia y none hace que no se defina*/
        "...";

Propiedades del hijo de Grid

Info 
                    
    grid-area: <name>;
    justify-self: start | end | center | stretch;
    align-self: start | end | center | stretch;

Codigo Avanzado de Grids

Info

Contenedor

                    
    grid-template: rows / columns   /* combina grid temmplate rows, columns y area */

Hijo

                    
    grid-area: <row-start>/<column-start>/<row-end>/<column-end>;

Animaciones

Info 
                    
        animation-name: nombre;
        animation-duration: #s (tiempo en segundos);
        animation-timing-function: ease (this is default) | linear | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n);
        animation-delay: #s;
        animation-iteration-count: infinite (o un #);
        animation-direction: normal | reverse | alternate | alternate-reverse;
        animation-fill-mode: 	forwards | backwards | both | none;
        animation-play-state: paused | running;
    
        @keyframes nombreAnimación {
            0%{alguna cosa}
            #%{alguna cosa}
            100%{alguna cosa}
        }
    
        o
    
        @keyframes nombreAnimación {
            from{alguna cosa}
            to{alguna cosa}
        }
        /* el from equivale a 0% y el to equivale a 100% */

Media queries

Info

Siempre pongan los medias al final del CSS. Si usan max, lso medias lo hacen en calores decrecientes y si usan min en valores crecientes. Pueden usar cualquier tipo de unidad de tamaño pero recomiendo fuertemente px.

                    
    @media (max-width: #px){
        .clases o id o elementos, etc{
            cambios a esta resolución
        }
    }
    @media (min width: #px){}
    @media (min-width: #px) and (max-width: #px){}
